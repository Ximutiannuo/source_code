## 系统简介

本系统是项目部的**数字化协作中心**，集成了总体计划、专项计划、日报填报、AI 智能助手、OCR 识别、自动工效核算等功能，支持以「数据驱动」替代「经验驱动」，实现项目部与各部门、各分包商的高效协同。亮点包括：**AI 自然语言提问**（无需翻报表，一句话查数据）、**OCR 现场单据识别**（拍照即录入）、**工效自动核算与预警**、**HMD 问题闭环追踪** 等，旨在让数据流动、让管理减负。

---

## 一、核心功能概览

| 模块 | 说明 |
|------|------|
| **总体计划** | 作业清单、甘特图、确认完成、批量关闭；与 P6 同步，作业新增/删除在 P6 维护 |
| **专项计划** | 月滚动、三月滚动计划；25 号封账；作业池领取、周期计划量分配；HMD 问题评分 |
| **日报填报** | 人力日报（MPDB）、工程量日报（VFACTDB）、验收日报（ITP/RFI） |
| **AI 助手** | 自然语言提问，自动检索数据并生成分析；支持完成量、人力、工程量、进度、工效等查询 |
| **OCR 识别** | 上传施工单据、验收单、报表照片，提取文字与表格，复制或导出后录入，减少手动输入 |
| **工效核算** | 根据人力与工程量自动推算工效，生成趋势曲线，提前预警异常 |

---

## 二、系统特色

* **智能 AI 助手**：无需翻找报表，用自然语言提问即可。例如「C01 本周完成量多少？」「各专业人力分布？」「月滚动计划各专业计划 vs 实际？」「EC1 混凝土工效多少？」。AI 将自动检索 VFACTDB、MPDB、VolumeControl、滚动计划、工效等数据并给出结果。
* **OCR 识别**：支持通用文字、表格、PDF 三种模式；可复制或导出 TXT/CSV/XLSX/Word，用于现场单据快速录入，减轻一线录入压力。
* **P6 计划同步**：作业与计划参数从 P6 自动同步（约 5 分钟）；作业新增、删除、调整需在 P6 中操作。
* **工效自动核算**：根据填报的人力与工程量实时推算当日工效，自动生成趋势曲线，提前预警异常。
* **HMD 问题评分**：专项计划中嵌入「How's My Driving」评价机制，计划编制方可对支持方（设计、采购、移交等）的响应与解决效果进行跟踪与评价，形成闭环。
* **问题追踪**：支持全员参与的问题与意见交流；异常可第一时间推送给责任人，确保事事有回音。

## 三、快速开始

1. **登录**：使用管理员分配的账号登录。
2. **筛选**：通过顶部筛选器选择子项目、主项、子项、装置、分包商等，缩小数据范围。
3. **日报**：进入人力日报 / 工程量日报 / 验收日报，按作业填报数据。
4. **专项计划**：进入专项计划，从作业池添加作业，分配周期计划量；遇外部制约可发起 HMD 问题。
5. **AI 助手**：在仪表盘或侧边栏打开 AI，用自然语言提问；问题越具体（带分包商、时间、位置），结果越精准。
