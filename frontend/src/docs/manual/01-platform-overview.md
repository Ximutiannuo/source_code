# 机械制造数字化平台概览

## 平台定位

本平台面向机械制造企业，尤其适用于以下场景：

- 非标定制比例高
- BOM 层级深、版本多
- 工艺路线与工位资源经常变化
- 订单交付需要打通设计、采购、制造、质量和设备

平台不再沿用原工程建设计划管理的业务主线，当前统一围绕机械制造交付链条建设：

`设计资料 -> BOM/工艺 -> 采购齐套 -> 工单执行 -> 质量放行 -> 设备保障 -> 交付复盘`

## 当前主导航

### 1. 制造主线

- 制造驾驶舱
- 制造订单与工单
- 物料主数据
- BOM 与产品结构
- 工艺模板与路线
- 采购与齐套

### 2. 质量与协同

- 质量管理
- 设备管理
- 工位与设施
- 焊接数据接口
- 设计资料接口
- 图纸 OCR

### 3. 平台治理

- 账户与角色
- 部门组织
- 系统维护

## 当前建设原则

### 数据一条线

- BOM 必须直接驱动物料需求和采购建议
- 工艺模板必须直接驱动工单与工序展开
- 工位报工、质检、停机必须自动反馈订单进度和执行状态

### 版本受控

- 设计版本、BOM 版本、工艺版本、工单执行版本需要可追溯
- ECN 不能停留在审批记录，必须继续影响采购、在制和计划

### 先闭环，再扩展

优先打通：

1. 订单主线
2. BOM 与工艺绑定
3. 采购齐套与制造执行联动
4. 工序级质量与设备数据回写

在主线稳定后，再继续建设 APS、成本核算、AI 辅助排产、视觉质检等增强能力。
