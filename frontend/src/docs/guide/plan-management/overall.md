# 总体计划管理：全局视野

总体计划管理由 **计划管理部** 负责核心维护，展示项目的整体进度脉络。

---

## 一、可以干什么

### 1.1 查看与浏览

* **甘特图**：通过横轴时间轴直观查看各作业的计划时间、实际时间及关联关系。
* **作业清单**：表格形式展示作业代码、描述、状态、进度、完成量等。
* **作业详情**：点击某作业可展开详情面板，查看关键工程量、非关键工程量、工程量及完工信息、人力投入等；**人力日报**、**工程量日报**的历史数据也可在详情面板中查看。在**特定权限**下（施工工程量、验收工程量、ABD工程量、OVR工程量信息管理的更新权限），可针对每条作业在本页面直接编辑工程量信息（上述四类工程量控制表）。

### 1.2 计划部操作（需权限）

* **计划参数调整**：作业层级、作业分类码（Activity Codes）等计划参数在 **P6** 中维护，通过同步将数据同步到本计划管理系统。
* **作业管理**：添加、删除作业，调整作业顺序或逻辑关系，均在 **P6** 中操作，同步后生效。
* **确认完成**：在本页面将作业标记为「已完成」；**完成后作业被锁定**，人力日报、工程量日报将禁止再填报该作业的数据。
  * **能关闭**：① 有预估总量且完工比例 ≥ 99.5%（99.5%～100% 间可补齐至 100% 或直接关闭；≥100% 直接关闭）；② 预估总量为 0 但有实际完成量；③ 纯人力作业（无预估总量且无完成量）。
  * **不能关闭**：有预估总量但完工比例 < 99.5% —— 需继续填报剩余工程量或修正预估总量。
* **重新打开**：仅对**已完成的作业**有效；解除锁定后允许继续填报人力与工程量数据。
* **批量关闭**：导出批量关闭模板（Excel），在确认列填写 CLOSE 后导入，系统逐条校验后批量关闭。**能关闭**的条件与「确认完成」相同；**不能关闭**的情形包括：作业不存在、无权限、完工比例 < 99.5%。

### 1.3 新增/调整/删除作业

如需**新增、调整或删除作业**，请联系**计划管理部**同事在 **P6** 中进行操作。系统会定期从 P6 同步数据到本计划管理系统，**约 5 分钟后**即可在作业清单中看到更新。

### 1.4 普通用户

* 通常只有查看权限，请通过**筛选器**定位到您关注的区域（分包商、子项目、主项、子项等）。

## 二、页面操作说明

### 2.1 筛选器

页面顶部的**筛选器**（分包商、子项目、主项、子项、装置、区块等）用于缩小数据范围。按层级依次选择：子项目 → 开车阶段 → 装置 → 区块 → 主项 → 子项 等，可快速定位到您负责的区域。筛选结果会同步到甘特图与作业表格。

### 2.2 分组

* 点击**分组**按钮，展开分组面板。
* 在「选择分组字段」中勾选一个或多个字段（如专业、工作包、子项、分包商等），表格将按所选字段逐层分组展示。
* 分组行可点击折叠/展开；支持「全部展开」「全部折叠」「展开到第 N 层」。

### 2.3 栏位设置

* 点击**栏位设置**按钮，打开栏位配置弹窗。
* 勾选需要显示的列（如作业代码、描述、状态、主项、子项、工作包、分包商、完成量等），取消勾选则隐藏该列。
* 可拖动列边界调整列宽；配置会随视图一并保存。

### 2.4 保存视图

* 点击**视图管理**按钮，打开视图列表。
* **保存当前视图**：将当前的分组方式、栏位显示、列宽保存为一个命名视图，便于下次快速恢复。
* **加载视图**：在列表中点击已保存的视图，即可应用该视图的分组与栏位配置。
* 视图保存在浏览器本地，刷新页面后仍可使用。
