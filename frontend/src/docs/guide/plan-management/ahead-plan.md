# 专项计划编制：目标承诺

专项计划是分包商对项目部的**完工承诺**，旨在把大计划落实到具体的每周执行上。本页面用于编制、审核、批准月滚动计划、三月滚动计划或自定义专项计划，并为每条作业分配各周的计划量（周期计划量）。

---

## 一、封账机制：25 号红线

* **分包商填报期**：每月 **1 号至 25 号**。在此期间，分包商可自由修改计划量、备注、作业描述，可从作业池加入、删除、导入计划。
* **自动封账点**：**每月 26 日 00:00**。封账后：
  * **普通用户**：无法修改计划量、备注、作业描述；无法删除、从作业池加入、导入；**意见**与**问题**仍可正常使用。
  * **仅系统管理员** 可变更计划。
* **特殊调整**：封账后如需变更，需报请计划管理部联系系统管理员进行修改。

---

## 二、责任人

| 角色 | 职责 |
|------|------|
| **分包商计划管理员 / 施工主管 / 计划部** | 编制计划：从作业池添加作业、分配周期计划量、填写备注与作业描述，发起问题与意见交流 |
| **施工主管** | 审核、批准、撤回；范围受子项目约束 |
| **施工管理部 / 计划管理部** | 审核、批准；计划经理、计划管理员、系统管理员可操作 |

## 三、页面顶部工具栏（逐项说明）

### 3.1 日期范围选择器

* **作用**：选择计划覆盖的起止日期。日期以**周五～周四**为一周，系统会根据选中范围自动推算各周周四日期，作为周期计划量的列头。
* **默认**：上月最后一个周五～当月最后一个周四。
* **联动**：变更日期后，`type_of_plan` 会更新为「计划类型_开始日期~结束日期」，影响数据加载与导入导出。

### 3.2 计划类型选择器

* **内置类型**：月滚动计划、三月滚动计划、春季攻势专项计划。
* **自定义计划**：在下拉底部输入框输入新名称，按回车保存为模板并选中。自定义名称会持久化到本地，下次仍可选。
* **与数据库**：内置类型对应中文前缀（如「月滚动计划」），自定义计划用用户输入的名称，均与日期拼接为完整 `type_of_plan`。

### 3.3 计划版本下拉

* **作用**：从已有计划版本（`plan_types`）中快速选择，自动填充日期和计划类型。
* **格式**：如「月滚动计划_2026-01-30~2026-02-26」。
* **清空**：手动修改日期或计划类型后，此下拉会自动清空，避免版本与当前展示不一致。

### 3.4 汇总 / 返回按钮

* **汇总**：切换到**汇总统计视图**，按工作包/资源/主要工作项/项目编码分组，查看周期计划量及审批进度；支持按周或按月钻取，支持对比实际完成。
* **返回**：从汇总视图切回主表格（甘特/列表）。

### 3.5 作业代码、作业描述筛选框

* **作用**：模糊搜索，缩小当前页展示的作业范围。搜索条件会传至服务端，支持命中不在首屏的作业。
* **作业代码**：按 `activity_id` 匹配。
* **作业描述**：按 `title`、`remarks`、`user_defined_activity_name` 匹配。

### 3.6 刷新按钮

* **作用**：从服务器拉取最新列表数据，用于多人协同下及时看到他人修改。

### 3.7 加载更多按钮

* **条件**：首屏加载 2000 条，若总数更多则显示「加载更多（已加载 x/总数）」。
* **作用**：点击后追加下一页数据，不替换已有数据。

### 3.8 分组按钮

* **作用**：展开或收起**分组面板**。
* **分组面板**：多选分组字段（专业、工作包、子项、分包商、执行阶段、项目、子项目、开车阶段、装置、主项、区块），表格按所选字段逐层分组；分组行可折叠/展开。

### 3.9 更多菜单

| 菜单项 | 作用 | 禁用条件 |
|--------|------|----------|
| **栏位设置** | 打开栏位配置弹窗，勾选要显示的列 | 汇总视图时禁用 |
| **视图管理** | 打开视图列表，保存/加载/删除视图 | 汇总视图时禁用 |
| **全部展开** | 将所有分组行设为展开 | 汇总视图时禁用 |
| **全部折叠** | 将所有分组行设为折叠 | 汇总视图时禁用 |
| **导入** | 触发文件选择，导入 Excel | 汇总视图或 25 日后（非管理员）禁用 |

### 3.10 删除选中按钮

* **作用**：批量从计划中删除选中的作业，这些作业将回到作业池。
* **禁用**：未选中作业、汇总视图、25 日后（非管理员）。
* **前置条件**：所选作业**不能包含已审核或已批准**的记录，否则提示取消勾选后再试。
* **确认**：点击后弹出确认框，确认后执行批量删除。

### 3.11 作业池按钮

* **作用**：打开「从作业池选取」弹窗，选择作业加入当前计划。
* **禁用**：汇总视图、25 日后（非管理员）。
* **详见**：下文「作业池弹窗」。

### 3.12 导出按钮

* **作用**：导出当前计划为 Excel，包含可见列 + 备注列，按当前筛选条件导出。
* **文件名**：`AheadPlan_计划类型_开始日期_结束日期.xlsx`。
* **禁用**：汇总视图时禁用。

### 3.13 审核选中、批准选中、撤回审核选中、撤回批准选中

* **可见条件**：仅施工主管、计划经理、计划管理员、系统管理员可见。
* **审核选中**：将选中作业的计划记录标记为「已审核」；审核后计划量、备注、作业描述**不可编辑**（可在意见栏交流）。
* **批准选中**：将选中作业的计划记录标记为「已批准」；须先审核，批准后同样锁定编辑。
* **撤回审核选中**：撤销选中作业的审核状态；仅审核人本人或更高层级用户可成功撤回。
* **撤回批准选中**：撤销选中作业的批准状态；仅批准人本人或更高层级用户可成功撤回。

## 六、主表格（甘特/列表）——列与操作

### 6.1 勾选列（第一列）

* **作用**：勾选作业，用于批量删除、审核、批准。
* **点击行为**：
  * **普通点击**：仅选中当前作业。
  * **Ctrl/Cmd + 点击**：切换该作业的选中状态（加入或移除）。
  * **Shift + 点击**：区间选择（从上次点击到当前作业之间的所有作业）。

### 6.2 基础列（状态、作业代码、作业描述等）

* **状态**：蓝块=已完成，蓝绿渐变=进行中，绿块=未开始。
* **作业代码**：只读。
* **作业描述（工程师填写）**：可编辑，防抖 1.5 秒自动保存；25 日后或已审核/批准时只读，悬停提示「当月 25 日后仅管理员可修改」或「已审核/已批准状态下不可修改」。
* 其它列（总量、完成量、剩余量、子项、专业、工作包等）为只读展示。

### 6.3 周期计划量列（每周一列）

* **列头**：每周的周四日期（YYYY-MM-DD）。
* **可编辑条件**：未审核且未批准，且未过 25 号（或为管理员）。
* **编辑方式**：直接输入数值，停止输入 1.5 秒后自动保存；`onBlur` 时立即保存。
* **锁定提示**：已审核/批准时显示「已审核/已批准状态下不可修改计划量，可在意见栏交流」；25 日后显示「当月 25 日后仅管理员可修改计划量」。
* **0 值展示**：未填写时显示为空，不显示 0。

### 6.4 备注列

* **可编辑**：与周期计划量相同（未审核且未批准，且未过 25 号或为管理员）。
* **防抖**：1.5 秒自动保存。
* **锁定**：同周期计划量。

### 6.5 问题列 —— HMD「How's My Driving」评分系统（创新点）

专项计划嵌入了 **HMD**（How's My Driving）评分系统，借鉴先进企业的评价理念：让计划编制方（分包商/施工主管/计划部）对执行支持方（设计、采购、仓储、移交等）的响应与解决效果进行**全程跟踪与评价**，形成闭环。

#### 核心理念

* **谁在承诺**：分包商/施工主管对项目部承诺「本周期完成多少」。
* **谁在支持**：图纸、材料、设备、移交条件等由设计、采购、仓储等部门提供。
* **谁在评价**：承诺方（提出问题的分包商/计划编制人）有权对支持方的解决效果进行 1～5 星评价。

HMD 把「承诺—障碍—解决—评价」串联起来，让执行障碍可见、可追踪、可量化，促进支持部门及时响应。

#### 页面入口与展示

* **无问题时**：显示「+」按钮，点击打开问题弹窗，可发起新问题。
* **有问题时**：显示徽章「未解决数/总数」，颜色表示健康度：
  * **红色**：有逾期未解决（需优先关注）；
  * **黄色**：有未解决但未逾期（在计划解决日期前）；
  * **绿色**：全部已解决（可进行确认评价）。
* **点击**：打开该作业的 HMD 问题弹窗，可查看、编辑、回复、解决、评价。

#### 问题类型（HMD 覆盖场景）

| 类型 | 说明 |
|------|------|
| 设计与技术问题 | 图纸、技术澄清、设计变更 |
| 采购与材料问题 | 材料未到、规格不符 |
| 仓储与设备问题 | 设备进场、仓储调配 |
| 施工管理问题 | 工作面、工序协调 |
| 安全环保问题 | HSE 审批、安全条件 |
| 质量管理问题 | 检验、验收条件 |
| 协调与接口问题 | 与其他专业/分包商的接口 |
| 审批与流程问题 | 流程卡点 |
| 人力与资源问题 | 人员、机具不足 |
| 其他 | 无法归类的障碍 |

#### 问题流程

1. **发起**：选择类型、优先级、解决部门、计划解决日期，可指定责任人。
2. **处理**：责任人/解决部门处理，可发布进展回复；解决后填写解决方案并标记为「已解决」。
3. **确认评价**：**提出方**（发起问题的人）确认问题已解决，并给出 1～5 星评价；评价后问题进入「已确认」状态，闭环完成。
4. **逾期提醒**：系统按计划解决日期计算「剩余天数」；逾期后显示「已逾期 x 天」，红色警示。

#### 消息与协作

* **@提及**：回复中可 @用户，被提及者会收到通知。
* **消息中心**：在主导航可查看「我的问题」「@我的问题」及未读通知，快速跳转处理。

### 6.6 意见列

* **展示**：消息图标，有意见时显示数量徽章。
* **点击**：弹出意见浮层，可查看历史意见、添加回复；支持 @提及。
* **可编辑**：不受审核/批准/25 号限制，随时可添加意见。

### 6.7 审核列

* **未审核**：施工主管/计划管理员可见「审核」按钮，点击后标记为已审核。
* **已审核**：显示「已审核」标签及审核人、时间；审核人本人或更高层级用户可见「撤回」按钮。

### 6.8 批准列

* **待审核**：显示「待审核」，不可批准。
* **已审核待批准**：计划经理/施工主管可见「批准」按钮。
* **已批准**：显示「已批准」标签及批准人、时间；批准人本人或更高层级用户可见「撤回」按钮。

## 七、作业池弹窗

### 7.1 打开方式

点击工具栏「作业池」按钮。25 日后非管理员时该按钮禁用。

### 7.2 筛选

* **按作业代码或描述筛选**：输入框模糊过滤作业池列表。
* 列表受全局筛选器（分包商、子项目、主项等）约束。

### 7.3 推荐作业

* **加载推荐**：点击后请求系统推荐作业（近 30 天有 MP/VFACT 报送且完成比例≠100% 的作业）。
* **勾选推荐**：在推荐加载完成后，一键勾选所有推荐且未关闭的作业。
* **勾选全部**：勾选当前列表中所有可加入的作业（排除已关闭）。
* **取消全部**：取消所有勾选。

### 7.4 作业列表

* **表格列**：作业代码、作业描述、工作包、分包商、总量、完成量、剩余量。
* **行勾选**：支持多选；**已关闭**的作业不可勾选（勾选框禁用，提示「已关闭的作业不可加入计划」）。
* **推荐标识**：推荐作业显示「推荐」标签；已关闭显示「已关闭」标签。

### 7.5 加入本月计划

* **操作**：点击「加入本月计划」确认。
* **逻辑**：为每条选中的未关闭作业，在计划周期内**每周创建一行**，计划量初始为 0。
* **分批**：每批 400 条，多批顺序请求。
* **结果**：提示「已加入：新增 x 条，已存在跳过 y 条」；关闭弹窗并刷新列表。

## 八、汇总统计视图

### 8.1 进入方式

点击工具栏「汇总」按钮。

### 8.5 表格结构

* **分组列**：分组名称、总量、完成量、剩余量。
* **周期计划量列**：各周/月计划量及占比。
* **审批状态列**：作业数、已审核数、已批准数及占比。
* **返回**：点击「返回」回到主表格。

## 九、导入与导出

### 9.1 导出

* **触发**：点击「导出」按钮。
* **格式**：Excel，包含当前可见列 + 备注列；按当前日期范围、计划类型、筛选条件导出。
* **用途**：离线填写周期计划量后，再通过导入更新。

### 9.2 导入

* **触发**：更多 → 导入，选择 .xlsx 文件。
* **确认**：弹窗显示即将导入到的 `type_of_plan`，需与当前页面一致，确认后执行。
* **格式要求**：
  * 第 4 行为列名行，需包含 `activity_id`；可选 `remarks`、`user_defined_activity_name`；日期列格式为 YYYY-MM-DD。
  * 第 6 行起为数据行；遇「签字」「Sign」等则停止解析。
* **逻辑**：按 activity_id 匹配，有则更新计划量与备注/作业描述，无则新建。
* **跳过规则**：已审核或已批准的记录**不更新**，仅跳过，导入完成后可查看「导入跳过详情」弹窗。
* **分批**：每批 200 条，显示进度。

## 十二、编制流程（简要）

1. **作业领取**：点击「作业池」，可选「加载推荐」→「勾选推荐」，或手动勾选，点击「加入本月计划」。
2. **计划量分配**：在周期计划量列填入各周承诺完成的数值；系统自动汇总为「周期计划量」并核算对整体进度的贡献率。
3. **备注与描述**：填写备注、作业描述（工程师填写），便于审核与追溯。
4. **协同**：若遇外部制约，在「问题」列发起问题；在「意见」列与审核人交流。
5. **审核与批准**：施工主管/计划管理员审核，计划经理/施工主管批准；批准后计划量、备注、作业描述锁定。
6. **导出/导入**：支持导出 Excel 离线填写后导入，已审核/批准行会跳过更新。
