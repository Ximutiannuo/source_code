# 人力日报：每日投入追踪

人力日报记录的是施工现场的**资源投入**。准确、详尽地填报人力数据，不仅是日常管理的需要，更是积累公司「组织过程资产」与「经验数据库」的核心举措。这些底层数据将用于优化劳动定额、提升施工生产力，并为后续项目的成本核算与进度预测提供科学依据。

---

## 一、责任人：由谁来填报人力日报

### 1.1 分包商计划管理员

* 负责**本单位（分包商）**范围内的人力日报填报。
* 有权录入、修改本单位今日的人力与机械投入数据。
* 不能操作其他分包商的数据。

### 1.2 子项目 / 计划部管理人员

* 负责**本子项目**或**全项目**范围内的日报管理。
* 有权查看、修改该范围内所有分包商填报的人力日报。
* 可对分包商填报内容进行审核、修正或删除。

### 1.3 施工部核查责任

施工部有责任和义务检查分包商的人力填报。如发现填报异常（如虚报、漏报、严重失实等），请及时反馈至**计划管理部**。计划管理部将对该分包商进行追责问责，并根据情节处以相应处罚，**根据合同约定最高可驱逐出场**。

---

## 二、填报要求

| 项目 | 说明 |
|------|------|
| **填报人** | 分包商计划管理员 |
| **截止时间** | **每日 12:00 前**（逾期将影响全项目的工效自动核算） |
| **填报对象** | **今日**预计投入的人力与机械 |
| **页面入口** | 菜单 → 日报填报 → **人力日报** |

---

## 三、核心流程：从选作业到保存

### 3.1 页面入口

菜单 → 日报填报 → **人力日报**

### 3.2 作业的含义

人力日报与**作业**一一关联。每条作业代表：

> **哪个分包商，在哪套图纸下，执行哪一个工作包的工作。**

* **分包商**：即施工承包商（如 C01、C02）。
* **图纸**：通过作业的子项、装置、主项等维度定位。
* **工作包**：即工程量类型（如焊接、混凝土等）。工作包定义可在 菜单 → **工作包管理** 中查阅。

### 3.3 筛选作业并填报

1. 在人力日报页面，通过顶部**筛选器**（分包商、子项目、主项、子项等）筛选出需要填报的作业。
2. 表格中每行对应一条**作业**，右侧为填报列：**人力**、**机械**、**备注**。
3. 直接在表格单元格中输入数值，无需弹出表单。
4. 填写完成后，点击**保存**按钮提交数据。

### 3.4 页面操作说明

#### 筛选器

页面顶部的**筛选器**（分包商、子项目、主项、子项、装置、区块等）用于缩小数据范围。按层级依次选择：子项目 → 开车阶段 → 装置 → 区块 → 主项 → 子项 等，可快速定位到您负责的区域。建议分包商用户先选定本单位（分包商）。

#### 分组、栏位、视图管理

* **分组**：点击「分组」按钮，选择分组字段（如专业、工作包、子项、分包商等），表格将按所选字段逐层分组；分组行可折叠/展开。
* **栏位设置**：点击「栏位设置」可勾选/取消需要显示的列。
* **视图管理**：可将当前分组方式、栏位配置、列宽保存为命名视图，下次可快速加载。

#### 导入与导出

| 操作 | 说明 |
|------|------|
| **导出 Excel** | 点击「导出 Excel」可导出当前筛选结果；**导出时需在筛选器中指定单一分包商**，以生成该单位的专属报表。 |
| **导入回写** | 点击「导入回写」，选择与系统导出一致格式的 Excel 文件，可将人力、机械、备注批量导入；导入前请确保格式与导出一致。 |

### 3.5 需要填写的字段

| 字段 | 说明 | 是否必填 |
|------|------|----------|
| **人力** | 该作业今日投入的总工人数（或工时） | 选填 |
| **机械** | 该作业今日占用的机具数量 | 选填 |
| **备注** | 现场异常说明，如停工、待料、极端天气等；是后续效率偏差分析的关键依据 | 选填 |

## 四、直接人力、辅助人力与间接人力

### 4.1 直接人力（填报到作业）

* **直接投入人力**：能明确归属到某一具体作业的人力，直接在该作业的「人力」列填报。
* **辅助人力**：现场无法直接定位在某一个具体工作上的人力，如通用脚手架工、机具加油、文明施工等。此类人力需填报到对应的工作包作业（CO 系列），区分如下：

| 工作包 | 说明 |
|------|------|
| **CO01** | 现场无法区分专业的人力（通用辅助） |
| **CO02** | 临设、场外辅助人力 |
| **CO03** | 偏向土建的辅助人力 |
| **CO04** | 偏向安装的辅助人力 |

* **建议**：凡是能够填报到具体作业的人力，建议填报到该作业，以便更准确反映人力投入。

### 4.2 间接人力（MP 额外项）

* **间接人力**：对应**MP 额外项**，如管理、技术、HSE、后勤、请假等，无法归属到具体作业的人力。
* 填报方式：在人力日报页面点击「**MP 额外项**」，在弹出的表单中按项（如「间接人力（工作）」「间接人力（请假）」等）填写数值。
* 填报 MP 额外项时，需在顶部筛选器中选定**单一分包商**，否则无法保存。

---

## 五、现场备注的重要性

如有停工、待料、极端天气或任何导致施工效率异常的情况，请务必在**备注**列详细记录。这些信息是后续进行效率偏差分析、区分「客观原因」与「管理问题」的关键依据。

---

## 六、工效自动核算

系统会根据您填报的「人力投入」与工程量日报中的「产出战果」，自动计算当日及累计的**工效（Productivity）**。您可随时在**工效分析**模块查看施工效率是否达到标准定额要求。

---

## 七、注意事项

* 系统已按权限限制可见数据，您只能看到并操作本单位（分包商）范围内的作业。
* 数据应尽可能真实地反映每个作业下的实际人员分布。
* 导出或填报 MP 额外项时，需在筛选器中指定**单一分包商**，以生成该单位的专属报表或保存额外项。
