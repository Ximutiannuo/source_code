# 智慧助手：AI 与 OCR 工具

系统集成了 **AI 智能助手** 与 **OCR 识别工具**，辅助数据分析与现场单据录入，提高生产力。

---

## 一、AI 助手（Chat Interface）

### 1.1 入口与用法

* **入口**：主导航 → 仪表盘，或侧边栏 AI 入口。
* **用法**：用自然语言提问，AI 将自动调用后台工具检索 VFACTDB、MPDB、VolumeControl、ahead_plan、productivity_cache 等数据，生成分析报告。
* **追问**：支持上下文追问，如先问「本周 C01 完成量」，再问「那上周呢」，AI 会延续上一问的筛选条件。

### 1.2 可提问的问题范围

基于当前已接入的 AI 工具，您可提问以下类型的问题（以下为示例，可按维度、时间、分包商、位置等自由组合）：

| 类别 | 示例问题 | 说明 |
|------|----------|------|
| **实际完成量** | C01 和 C02 本周的填报完成情况如何？<br>钢结构累计完成多少？<br>ECU 子项目昨天完成了多少？ | 来自 VFACTDB，与滚动计划无关；支持按分包商、子项目、装置、专业等分组 |
| **人力投入** | PI01 昨天投入多少人？<br>各专业人力分布？<br>C13 本周工日是多少？ | 来自 MPDB；支持按专业、分包商、子项、装置等分组 |
| **工程量控制** | 某分包商工程量总量情况？<br>各工作包分别的量？<br>12510-01 的施工完成和验收进度？ | 总量、材料到货、工作面、施工完成、验收、竣工、收款；「本周实际完成」用实际完成量工具 |
| **滚动计划** | 月滚动计划各专业计划 vs 实际？<br>C01 本周计划完成多少？<br>各子项目计划 vs 实际对比？ | 需指定计划类型（如 月滚动计划_2026-01-30~2026-02-26）；支持按子项目、分包商分组 |
| **计划提交/审批** | 钢结构月滚动计划都是谁提交/审批/批准的？ | 查询 ahead_plan 中的提交人、审批人、批准人 |
| **累计进度** | 项目累计进度（WF%）多少？<br>各专业/各子项/各装置进度？<br>设计/采购/施工阶段进度？ | 单维走缓存，组合走实时；支持 breakdown_by 按专业、子项、装置、主项、子项目、分包商 |
| **进度增量** | 本周/上月进度增量？ | 某时间段内 WF% 增量 |
| **工效** | EC1 混凝土的工效多少？<br>各分包商工效对比？<br>钢结构本周工效多少？ | 来自 productivity_cache；支持按分包商、子项目、子项、装置等分组；装置取值为 EC1、EC2、PE1、PE2 等 |

### 1.3 常用维度与取值

* **分包商**：C01、C02、C05、C07、C09、C12、C13、C15、C16、C17、C18、C19 或常用称呼；系统会自动解析并在结果中注明。
* **时间**：支持相对词（本周、上周、昨天、今天、全周期、累计）及日期范围（如 2025-01-01至2026-02-12）；**周** 定义为周五～周四。
* **位置/设施**：子项目（ECU、PEL、UIO）、装置（EC1、EC2、PE1、PE2）、主项（12200、12401）、子项、开车阶段（T0、T1、T2）、区块（Q11、Q21）等，取值以系统实际数据为准。
* **工程量类型**：钢结构、混凝土、焊接、土建、电仪、设备、PI01 等。

### 1.4 使用建议

* 问题尽量具体：带分包商、时间、位置或专业时，AI 能更精准筛选。
* 若返回无数据，可尝试放宽筛选（如不限定分包商）或核对维度取值是否与系统一致。

## 二、OCR 识别工具

### 2.1 入口

**实用工具** → **OCR 识别**。

### 2.2 功能模式

| 模式 | 适用场景 | 支持格式 |
|------|----------|----------|
| **通用文字识别** | 纯文本、照片、截图 | 图片（jpg/png/bmp/gif/webp）或 PDF（转图后 OCR） |
| **表格识别** | 表格、报表、规格表 | 同上；先检测网格线，若无框线则按行分组 |
| **PDF 提取** | 仅电子版 PDF | 仅 PDF；直接抓取文本与表格，不做 OCR。扫描版请用「通用文字识别」或「表格识别」 |

### 2.3 识别语言（通用/表格模式）

* 中英（通用）、纯英文、俄文（西里尔）、英俄、中俄。
* **噪点检查**：可开关，用于移除箭头、格式符号等干扰。

### 2.4 结果操作

* **复制**：一键复制到剪贴板。
* **导出**：TXT、CSV、MD、XLSX（含合并 / 扁平）、Word。
* **价值**：现场拍摄的施工单据、验收单或报表照片，可快速提取关键字段（作业代码、数值、日期等），复制或导出后录入日报，减少手动输入。

### 2.5 使用提示

* 图片建议清晰、正对拍摄，避免严重倾斜或模糊。
* 电子版 PDF 用「PDF 提取」最快；扫描版 PDF 建议用「通用文字识别」或「表格识别」。
